# ResearchArtifacts Data Schemas

[JSON Schema](https://json-schema.org/) definitions for all data structures produced by the [artifact_analysis](https://github.com/researchartifacts/artifact_analysis) pipeline and consumed by the [researchartifacts.github.io](https://github.com/researchartifacts/researchartifacts.github.io) website.

**Browse the documentation:** [researchartifacts.github.io/data-schemas](https://researchartifacts.github.io/data-schemas/)

## Schemas

| Schema | Description | Generated by |
|--------|-------------|--------------|
| [Artifacts](https://researchartifacts.github.io/data-schemas/artifacts.html) | Core artifact records with badges and URLs | Pipeline output |
| [Institution Rankings](https://researchartifacts.github.io/data-schemas/institution_rankings.html) | Institution-level rankings and metrics | `generate_institution_rankings.py` |
| [Combined Rankings](https://researchartifacts.github.io/data-schemas/combined_rankings.html) | Author rankings combining artifacts + AE service | `generate_combined_rankings.py` |
| [Author Stats](https://researchartifacts.github.io/data-schemas/author_stats.html) | Detailed per-author statistics and paper lists | `generate_author_stats.py` |
| [Search Data](https://researchartifacts.github.io/data-schemas/search_data.html) | Merged data for website full-text search | `generate_search_data.py` |
| [Repo Stats](https://researchartifacts.github.io/data-schemas/repo_stats.html) | Per-artifact repository metrics (GitHub/Zenodo/Figshare) | `generate_repo_stats.py` |
| [Summary](https://researchartifacts.github.io/data-schemas/summary.html) | High-level site summary statistics | `generate_statistics.py` |
| [Artifacts by Conference](https://researchartifacts.github.io/data-schemas/artifacts_by_conference.html) | Badge breakdown by conference and year | `generate_statistics.py` |
| [Artifacts by Year](https://researchartifacts.github.io/data-schemas/artifacts_by_year.html) | Year-over-year artifact counts | `generate_statistics.py` |
| [Repo Stats Summary](https://researchartifacts.github.io/data-schemas/repo_stats_summary.html) | Aggregated repository metrics for the site | `generate_repo_stats.py` |
